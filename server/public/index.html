NepalBooks Update Server

This server provides update information for the NepalBooks accounting application.

API Endpoints

GET /api/updates/latest/:channel

Get the latest release for a specific channel (stable or beta)

Example: /api/updates/latest/stable

GET /api/updates/releases/:channel

Get all releases for a specific channel

Example: /api/updates/releases/stable

GET /api/updates/version/:version

Get information for a specific version

Example: /api/updates/version/1.0.0

POST /api/updates/admin/auth

Authenticate as admin (requires password)

POST /api/updates/admin/publish

Publish a new release (requires authentication token)

Administration

For administration access, please use the NepalBooks application's console during development:

updateAdmin.showAdminPanel()

This will display a UI for publishing new releases and managing updates.
